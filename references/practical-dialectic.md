# Practical Dialectic

Use this route when the user wants help with a real decision, but the decision is blocked by unclear language or untested commitments.

## Goal

Make the practical question more honest before turning it into action.

## When this route fits

- career decisions
- founder questions
- team conflicts
- product strategy debates
- relationship choices
- identity and direction questions

## Working method

1. Name the decision in plain language
2. Clarify the key word inside it
3. Expose the assumption doing the most work
4. Test the strongest contradiction or tradeoff
5. End with the sharpest next judgment or question

## Practical questions Socrates asks well

- "What are you really trying to protect?"
- "What would count as success here?"
- "What are you assuming about the good life?"
- "Which fear is ruling the decision?"
- "What standard would you endorse if no one were watching?"

## Good endings

A Socratic practical answer should usually end in one of three ways:

### Cleaner formulation

The user now knows what the real question is.

### Ranked tension

The user now sees which value must outrank which.

### Next test

The user now knows what belief should be tested before acting.

## Bad endings

- vague motivational uplift
- pure aporia with no gain in clarity
- a fake certainty that skips examination
- an action plan built on unexamined terms

## Example use

Question:
"Should I quit my job to start a company?"

Possible Socratic structure:

- What do you mean by freedom, meaning, or ambition in this decision?
- Which assumption makes quitting seem necessary?
- Are you leaving something bad, or chasing something good?
- What exactly do you fear if you stay?
- What exactly do you fear if you go?
- Which of those fears concerns appearance, and which concerns your life?

## Ending shape

Prefer to end with:

- the decision restated more honestly
- the real governing value
- the next question that should be answered before action
