# Definition Clarification

Use this route when the user is using a large word as though it were already clear.

## Goal

Do not answer the question at the level of slogan.
Find the key term, test what the user means by it, and narrow the field of meaning.

## Typical trigger words

- freedom
- justice
- authenticity
- success
- truth
- courage
- responsibility
- intelligence
- alignment
- meaning

## Working method

1. Ask what the user means by the key term
2. Separate two or three candidate meanings
3. Test whether the user's later claims fit all of them or only one
4. Reject a definition that is too wide, too narrow, circular, or unstable
5. End with the cleanest working definition available so far

## Definition tests

### Test 1: Breadth

If the definition includes clearly bad or irrelevant cases, it is too wide.

### Test 2: Narrowness

If the definition excludes cases the user wants to keep, it is too narrow.

### Test 3: Circularity

If the word being defined quietly reappears in the explanation, the definition has not advanced.

### Test 4: Empty approval

If the definition is only praise in disguise, it is not yet a definition.

### Test 5: Actionability

If nothing would change in practice depending on the definition, the word is probably still foggy.

## Good question patterns

- "What do you mean by that word here?"
- "Would this case count too?"
- "If that is your definition, why exclude this example?"
- "Are you defining the thing, or only praising it?"
- "Does your definition still hold when the stakes change?"

## Bad habits to avoid

- Do not accept the user's first wording as stable
- Do not replace the word with a synonym and pretend the job is done
- Do not ask six questions at once
- Do not turn the exchange into vocabulary theater

## Example use

If the user says:
"I want freedom."

Possible senses may include:

- no boss
- no external coercion
- room to choose among options
- not being ruled by appetite or fear
- a life ordered by reasons one can endorse

The route should then test which of these the user actually means.

## Ending shape

Prefer to end with:

- the unstable original word
- the cleaner working definition
- the remaining unresolved border case
