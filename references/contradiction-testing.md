# Contradiction Testing

Use this route when the user holds two commitments that seem hard to keep together.

## Goal

Do not merely spot inconsistency for sport.
Show where two commitments collide, and what the collision reveals.

## Typical triggers

- "I want freedom, but I need approval"
- "I value honesty, but not if it costs too much"
- "I want deep work, but I also want constant visibility"
- "I want AI to think for me, but I also want my own judgment"

## Working method

1. State the first commitment
2. State the second commitment
3. Ask whether both can hold in the same case
4. Introduce one counterexample or consequence
5. Name the actual tension
6. End with a cleaner ranking, distinction, or unresolved conflict

## Forms of contradiction

### Logical contradiction

The two claims cannot both be true in the same sense.

### Practical contradiction

The user says both, but their action standard cannot satisfy both.

### Moral contradiction

The user's stated values conflict when cost enters.

### Identity contradiction

The kind of person they say they want to be conflicts with the method they endorse.

## Useful question forms

- "Can both of these be true at once?"
- "Would you still say this in the harder case?"
- "What happens when the cost appears?"
- "Which of the two do you abandon first in practice?"
- "If both cannot stand together, which one actually rules?"

## Counterexample discipline

- Use one good case, not many weak ones
- Prefer an example the user would recognize as relevant
- Avoid rhetorical overkill

## Failure modes

- turning all nuance into contradiction
- pushing the user into a false dilemma
- humiliating rather than clarifying
- exposing tension without offering a cleaner statement

## Example use

User says:
"I believe in radical honesty, but some truths are too disruptive to say."

The route should ask:

- What makes honesty radical here?
- Is your rule really honesty, or honesty unless the social cost rises?
- Are you defending discretion, kindness, or self-protection?

## Ending shape

Prefer to end with:

- the two claims
- the specific point of collision
- the cleanest revised formulation
